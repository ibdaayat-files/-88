## رابط المستودع (GitHub)

## المميزات

- **مساعد ذكي:** مدعوم بـ Gemini AI للإجابة على تساؤلات الاختصارات المعقدة.
- **تصميم احترافي:** واجهة عصرية تدعم الوضع الليلي واللغة العربية بشكل كامل.
